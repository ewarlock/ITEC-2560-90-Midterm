Dog Breed Browser

This tool can be used to view pictures of different dog breeds, and add breeds to a favorites list. This tool uses The Dog API to generate a picture for each breed, and uses Wikipedia to find information on each breed's temperament.

Please be aware that temperament information is not available for all breeds as listed. This is a limitation of the dog breed browser, and may be mitigated by searching for the breed directly on Wikipedia, or through other sources.

This webpage was created by Edwin Kibbons for ITEC-2560-90. Enjoy :)

Favorites

Click "Add to Favorites" when viewing a dog breed to add it to this list. Click the red minus button to remove a breed from the list.
